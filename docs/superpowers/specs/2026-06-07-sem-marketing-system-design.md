## 一、项目概述

为华创生电机（hcsmotor.cn）搭建百度 SEM + Bing Ads 双引擎广告投流系统。包含：竞品关键词策略、5 个产品专用落地页、全链路追踪系统、线索管理、部署方案。

### 前提条件

| 项目 | 状态 |
|------|------|
| 域名 `hcsmotor.cn` | ✅ 已部署 |
| HTTPS | ✅ 已有 |
| ICP 备案 | ✅ 已完成 |
| MySQL 数据库 | ✅ `motor_website` 库已运行 |

### 预算与平台

| 平台 | 预算占比 | 月预算 |
|------|---------|--------|
| 百度 SEM | 50% | ¥1,500-2,500 |
| Bing Ads | 50% | ¥1,500-2,500 |
| **合计** | **100%** | **¥3,000-5,000** |

> **为什么五五开：** 同等预算控制变量，跑 1-2 个月后直接对比 CPC、转化率、有效线索数，数据说话决定后续预算倾斜方向。360 暂时不投，精力集中在两个核心渠道上。

### 转化目标（双转化）

- 电话拨打（`tel_click`）
- 表单提交（`form_submit`）

## 五、追踪系统

### 架构

```
track.js（单文件，所有页面引用，~200行）

├── 广告来源解析
│   ├── 百度: ?bd_vid=xxx&keyword=xxx&plan=xxx
│   ├── Bing:  ?utm_source=bing&utm_campaign=xxx&utm_term=xxx
│   └── 解析后写入 localStorage，跨页面保持
│
├── 自动事件
│   ├── 页面加载 → ad_landing（含 platform、keyword）
│   ├── 滚动深度 50% / 75% / 100%
│   └── 停留时长 30s / 60s / 120s
│
├── 点击事件（CSS class 驱动，零侵入）
│   ├── [data-track="tel_click"]     → 电话点击
│   ├── [data-track="cta_click"]     → CTA 按钮
│   ├── [data-track="form_submit"]   → 表单提交
│   └── [data-track="exit_link"]     → 安全出口点击
│
└── 表单追踪
    └── 提交时自动注入广告来源隐藏字段
        (ad_platform, ad_keyword, ad_plan, ad_bd_vid, ...)
```

### 数据输出

| 百度统计可看 | 你们 MySQL 可查 |
|------------|---------------|
| PV / UV / IP | 客户姓名 / 电话 / 公司 |
| 来源关键词 | 广告来源（哪个词带来） |
| 停留时长 / 跳出率 | 落地页来源 |
| 转化率 / 漏斗 | 跟进状态 |

### 后期加 Google/Bing 追踪

track.js 中 `send()` 函数里加一行即可，所有调用点不变：

```javascript
// 百度（已有）
if (window._hmt) _hmt.push(['_trackEvent', name, ...]);

// Bing（后期加）
if (window.uetq) uetq.push('event', name, data);

// Google（后期加）
if (window.gtag) gtag('event', name, data);
```

## 八、部署方案

### 统一部署（推荐：Nginx 反代）

```
服务器（阿里云/腾讯云 ECS，2核4G）

Nginx（443 HTTPS + 80→443 重定向）
├── SSL: Let's Encrypt 免费证书
├── 静态文件: dist/* 直接服务
└── API 反代 → localhost:8000

FastAPI（localhost:8000，supervisor/systemd 守护）
├── 静态文件挂载
├── /api/products CRUD
├── /api/categories
├── /api/contact
├── /api/lead         ← 新增：接收落地页表单
├── /api/audit
└── /admin             → 管理后台

MySQL 8.0（同机或云数据库）
├── motor_website 库
│   ├── categories
│   ├── products
│   └── leads          ← 新增
```

### 部署流程

```
1. 服务器装环境：Python 3.10+, MySQL 8.0, Nginx, supervisor
2. git clone 代码到服务器
3. python build.py 构建 dist/
4. 配置 Nginx + SSL 证书
5. systemctl start motor-api
6. 百度统计 + 百度 SEM 账户配置
7. Bing Ads 账户配置
```
